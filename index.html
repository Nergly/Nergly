Commandable Plugin

A versatile plugin that allows server admins to create time-bound commands with dynamic abilities. Players can receive temporary powers or permissions, with notifications when their abilities expire. Perfect for adding a magical touch to any Minecraft server!

MrAnnouncer Plugin

MrAnnouncer integrates seamlessly with Discord, allowing your Minecraft server to broadcast messages or events to Discord channels. Keep your community informed with this automated announcement system!
